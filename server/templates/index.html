Vehicle Detection Demo

Our web application is designed to detect, track, and count vehicles using advanced algorithms. We employ the You Only Look Once (YOLO) algorithm for object detection and the DeepSORT algorithm for tracking. When a vehicle crosses a designated line, the system automatically creates a traffic data object and inserts it into our MongoDB Atlas database collection. This traffic data can be retrieved via GET requests to our API
